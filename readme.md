# Python + DeepSeek + Streamlit Chatbot App

## Installation

1. Install Python v3.11.6
```bash
python -m venv venv
```
2. Install virtual environment
```bash
python -m venv venv
```
3. Activate virtual environment
```bash
python -m venv venv
````
5. Install Dependecies from requirements.txt
```bash
pip install -r requirements.txt
````
6. Start Streamlit
```bash
stremlit run app.py
````
